# Поліморфізм в Python 3

### Мета роботи - засвоїти застосування принципу поліморфізму в об’єктно-орієнтованому програмуванні.

### Завдання
1. Ознайомитися з поняттям поліморфізму в ООП.
2. Навчитися перевизначати поведінку методів.
3. Реалізувати декілька “магічних методів” для роботи з визначеними
раніше класами.

### Хід роботи
1. Створюємо основний клас та перевизначаємо методи *repr, str, add, sub, len* у [файлі](./project.py).
2. Проводимо тестування роботи наших перевизначених методів.

### Висновок
Ми успішно використали принцип поліморфізму в об’єктно-орієнтованому програмуванні, перевизначивши поведінку методів.
